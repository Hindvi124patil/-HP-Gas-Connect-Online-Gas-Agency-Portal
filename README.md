# 🔥 HPGas Connect – Online Gas Agency Portal

## 📌 Description

HPGas Connect is a web-based application developed using HTML, CSS, and JavaScript. It allows users to book gas cylinders, track delivery, raise complaints, and view important information like pricing, safety, and services.

## 🎯 Features

* Gas Cylinder Booking
* Order Tracking System
* Complaint Registration
* Safety Guidelines
* Pricing Information
* Contact Page

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

* index.html (Home Page)
* booking.html
* tracking.html
* complaint.html
* contact.html
* services.html
* safety.html
* pricing.html
* css/
* js/
* images/

## 🚀 How to Run

1. Download or clone the project
2. Open index.html in browser
